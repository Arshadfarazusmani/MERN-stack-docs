# üß© Modules in Node.js

## üìò What is a Module?

A **module** in Node.js is simply a **reusable block of code** whose existence **does not accidentally impact** other code.
Node.js uses modules to organize and reuse code efficiently.

Every JavaScript file in Node.js is treated as a **separate module**.

---

## üèóÔ∏è Types of Modules

Node.js has **three types** of modules:

### 1. **Core (Built-in) Modules**

These come **pre-installed** with Node.js.

**Examples:**

| Module   | Purpose                          |
| -------- | -------------------------------- |
| `fs`     | File System (read/write files)   |
| `http`   | Create HTTP servers and clients  |
| `url`    | URL parsing and formatting       |
| `path`   | File and directory path handling |
| `os`     | Provides system-related info     |
| `events` | Handle events and emitters       |

**Example:**

```js
const fs = require('fs');
fs.writeFileSync('example.txt', 'Hello Node.js!');
```

---

### 2. **Local (User-defined) Modules**

These are **custom modules** created by developers to organize project code.

**Example:**
**math.js**

```js
function add(a, b) {
  return a + b;
}
module.exports = add;   // export the function
```

**app.js**

```js
const add = require('./math');
console.log(add(5, 3)); // Output: 8
```

---

### 3. **Third-Party Modules**

Installed using **npm (Node Package Manager)**.
These are community-built modules.

**Example:**

```bash
npm install express
```

**Usage:**

```js
const express = require('express');
const app = express();

app.get('/', (req, res) => res.send('Hello World'));
app.listen(3000);
```

---

## ‚öôÔ∏è Module Wrapper Function

Node.js internally wraps each module in a function before execution:

```js
(function(exports, require, module, __filename, __dirname) {
   // module code here
});
```

This allows each module to have its own **scope**.

---

## üì¶ Exporting and Importing

### Export:

```js
module.exports = { functionName, variable };
```

### Import:

```js
const { functionName, variable } = require('./moduleName');
```

## üß† Key Points

* Each file is a separate module.
* Use `require()` to import and `module.exports` to export.
* Node.js caches modules after the first load (for performance).
* Modules promote **code reusability** and **maintainability**.
