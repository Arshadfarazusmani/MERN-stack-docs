# Usecase 

## Access token 
```js
userSchema.methods.generate_Access_Token=  function () {
   return jwt.sign(
        {
            _id:this._id,
            email:this.email,
            username:this.username

        },
        process.env.JWT_ACCESS_TOKEN_SECRET,
        {
            "expiresIn":process.env.JWT_ACCESS_TOKEN_SECRET_EXPIRY
        }
    )

    
    
}
```
