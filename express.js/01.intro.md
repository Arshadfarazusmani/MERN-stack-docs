## üöÄ Creating Your First Express Server

```js
const express = require('express');
const app = express();

// Define a route
app.get('/', (req, res) => {
  res.send('Hello from Express.js!');
});

// Start the server
app.listen(3000, () => {
  console.log('Server running at http://localhost:3000');
});
```

---

## üß≠ Routing in Express

Routing is how an application responds to **client requests** (GET, POST, etc.).

```js
app.get('/', (req, res) => res.send('Home Page'));
app.get('/about', (req, res) => res.send('About Page'));
app.post('/login', (req, res) => res.send('Login Successful'));
```

> The **HTTP methods** (GET, POST, PUT, DELETE) are used to handle different actions.

---

## üß± Middleware in Express

Middleware functions are used to **process requests** before sending responses.

### Example:

```js
const express = require('express');
const app = express();

// Middleware example
app.use((req, res, next) => {
  console.log('Request URL:', req.url);
  next(); // passes control to next middleware/route
});

app.get('/', (req, res) => res.send('Middleware Example'));

app.listen(3000);
```

---

## üì¶ Serving Static Files

You can serve static files like HTML, CSS, or images easily.

```js
app.use(express.static('public'));
```

> This means files in the `public` folder can be accessed directly, e.g., `/public/style.css`.

---

## üíæ Handling JSON and Form Data

Express provides built-in middleware to handle request data.

```js
app.use(express.json()); // to handle JSON data
app.use(express.urlencoded({ extended: true })); // to handle form data
```

---

## üß† Express Routing Parameters

You can capture dynamic values in URLs.

```js
app.get('/user/:id', (req, res) => {
  res.send(`User ID: ${req.params.id}`);
});
```

---

## üß© Example: REST API using Express

```js
const express = require('express');
const app = express();
app.use(express.json());

const users = [{ id: 1, name: 'Arshad' }];

app.get('/users', (req, res) => res.json(users));  // Read
app.post('/users', (req, res) => {                 // Create
  users.push(req.body);
  res.status(201).send('User added');
});

app.listen(4000, () => console.log('API running on port 4000'));
```
